# price-tag Agent

多平台旅游价格查询助手

## 核心能力

### 1. 多平台价格查询

支持以下平台：
- **携程网** (ctrip.com)
- **去哪儿网** (qunar.com)
- **Agoda** (agoda.com)

### 2. 查询类型

- ✈️ 机票查询
- 🏨 住宿查询

### 3. 功能特点

- 自动化浏览器查询
- 多平台价格对比
- 数据提取和整理
- 生成比价报告

## 技术实现

### Skills依赖

#### browser skill
- **用途**：自动化浏览器操作，访问旅游网站
- **能力**：
  - 打开指定网站
  - 填写查询表单
  - 提取搜索结果
  - 截图和数据采集

### 工作流程

```
1. 接收查询请求（出发地、目的地、日期）
2. 依次访问各平台
3. 填写查询条件
4. 提取价格信息
5. 生成比价报告
```

## 使用方法

### 机票查询
```
查询北京到上海 2026-03-15 的机票价格
```

### 住宿查询
```
查询上海 2026-03-15 到 2026-03-17 的住宿价格
```

## 注意事项

- 查询时需要等待页面加载
- 不同平台可能需要不同的查询策略
- 价格实时变动，仅供参考
- 遵守网站使用条款

## 配置要求

- 需要browser skill支持
- 网络连接稳定
- 建议使用headless模式提高效率
