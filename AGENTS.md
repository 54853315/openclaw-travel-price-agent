# price-tag Agent

多平台旅游价格查询助手

## 核心能力

### 1. 多平台价格查询

支持以下平台：
- **携程网** (ctrip.com)
- **去哪儿网** (qunar.com)
- **Agoda** (agoda.com)

### 2. 查询类型

- ✈️ 机票查询
- 🏨 住宿查询

### 3. 功能特点

- 自动化浏览器查询
- 多平台价格对比
- 数据提取和整理
- 生成比价报告

## 技术实现

### 脚本文件

| 脚本 | 用途 | 触发场景 |
|------|------|----------|
| `agoda-persistent.js` | Agoda 酒店价格查询 | 用户查询酒店/住宿价格 |
| `ctrip-flight.js` | 携程国际机票查询 | 用户查询国际机票 |

### 查询行为规范

**Agoda 酒店查询** (`agoda-persistent.js`)：
- ✅ 所有 Agoda/酒店查询必须使用此脚本
- 脚本位置：`/Users/konakona/.openclaw/workspace/price-tag/agoda-persistent.js`

**调用方式（命令行参数）：**
```bash
node agoda-persistent.js --checkIn 2026-05-15 --checkOut 2026-05-17 --adults 2 --rooms 1
```

**参数说明：**
| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--checkIn` | 入住日期 (YYYY-MM-DD) | 2026-06-01 |
| `--checkOut` | 退房日期 (YYYY-MM-DD) | 2026-06-12 |
| `--adults` | 成人数量 | 3 |
| `--rooms` | 房间数 | 1 |

> 注意：城市 ID 已固定为图鲁斯社岛 (280892)，如需查询其他城市需修改脚本中的 URL。

**携程机票查询** (`ctrip-flight.js`)：
- 用于国际机票查询（马尔代夫、巴厘岛等）
- 脚本位置：`/Users/konakona/.openclaw/workspace/price-tag/ctrip-flight.js`
- 修改配置：`fromCode`, `toCode`, `date`, `adults`
- 运行命令：`node ctrip-flight.js`
- 结果保存到：`ctrip-results.md`

### Skills依赖

#### browser skill
- **用途**：自动化浏览器操作，访问旅游网站
- **能力**：
  - 打开指定网站
  - 填写查询表单
  - 提取搜索结果
  - 截图和数据采集

### 工作流程

```
1. 接收查询请求（出发地、目的地、日期）
2. 依次访问各平台
3. 填写查询条件
4. 提取价格信息
5. 生成比价报告
```
